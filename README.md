# agent-skills

A collection of skills for [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Installation

```bash
# Install globally
npx skills add hyf0/agent-skills -g

# Or install to a specific project
npx skills add hyf0/agent-skills
```

## Available Skills

### `/vscode`

Open the current working directory or a specified folder in Visual Studio Code.

**Usage:**
- `/vscode` - Opens the current working directory
- `/vscode /path/to/folder` - Opens the specified folder
